# Dashboard-de-vendas-em-Excel
Este projeto teve como objetivo demonstrar como os dashboards são essenciais para a tomada de decisões rápidas. O projecto foi desenvolvido com base as aulas assistidas no curso de Excel com Inteligência artificial, pelo professor Filipão. O dashboard é de fácil utilização e apresenta um visual amigável para aqueles que amam a uniformidade visual.

O projecto foi desenvolvido a partir de uma base de dados pronta e contou a utilização de várias técnicas e recursos aprendidos durante as aulas. Destaca-se a utilização de tabelas dinâmicas, com vista a resumir dados e apresentar resultados de forma rápida e simples. A par disto, foram também utilizados outros recursos que acompanham as tabelas dinâmicas, como os gráficos dinâmicos e segmentação de dados.

No que concerne a segmentação de dados, foi implementada uma nova coluna com fórmula que permitisse distinguir as vendas por periodos mensais, trimestral e anual, com as respectivas subcategorias.
